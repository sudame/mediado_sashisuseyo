# 開発（バックエンド）
# 初期設定
1: .仮想環境の構築
```
$ pip install vitualenv <ファイル名>
```
```
$ virtualenv <ファイル名>\Scripts\activate
```
2: .必要なライブラリのインストール
```
$ python -V
Python 3.8.0

& pip -V
pip 19.2.3
```
```
$ pip install flask
```

3: .ローカルサーバーの立ち上げ
```
python app.py
````
4: .herokuへのデプロイ
